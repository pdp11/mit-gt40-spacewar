Using SIMH to Install PDP-11 DOS Onto a RK05 Disk Using Installation Media

I used the documentation and install media that others have captured as the basis
to get DOS-11 v9-20C installed onto a RK05 disk image under SIMH version 3.8.
Once I have a disk prepared, I use my homebrewed transfer program that transfers
the disk image onto a real RK05 cartridge.
